## Screenshot Best Practices

1. **Clean status bar**: Use a device with full battery, no notifications
2. **Time**: Set to 9:41 AM (Apple convention) or a clean time
3. **Content**: Use meaningful Bible passages that demonstrate features
4. **Consistency**: Use same translation across screenshots
5. **Highlights**: Add some example highlights in different colors
6. **Dark mode**: Make sure to capture both light and dark modes

## Suggested Passages for Screenshots

- **Genesis 1:1** - Great for Hebrew interlinear
- **John 1:1** - Great for Greek interlinear ("In the beginning was the Word")
- **Psalm 23** - Beautiful, well-known passage
- **John 3:16** - Most recognized verse
- **Romans 8:28** - Popular for notes/highlights demo
- **Philippians 4:13** - Good for showing verse-by-verse view
